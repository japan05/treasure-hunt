# Treasure Hunt - Wumpus World

A Python-based game UI for the Wumpus World treasure hunting game.

## Features

- Clean, minimalist UI with light blue theme
- Main menu with Start and Settings buttons
- 3x6 game grid for Wumpus World gameplay
- Settings screen for configuring game parameters
- Settings icon using Flaticon

## Requirements

- Python 3.x
- Pillow (PIL) for image handling

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Game

```bash
cd python
python wumpus_ui.py
```

Or from the project root:
```bash
python python/wumpus_ui.py
```

## Settings Icon

The settings icon is from Flaticon. If you need to replace it:
1. Download a settings icon from [Flaticon](https://www.flaticon.com/free-icons/settings)
2. Save it as `images/settings_icon.png`
